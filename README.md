## Todos Node MongoDB Express

Make sure to have a MongoDB instance running on your system at port: 27017

###### Running Server

1) 

```bash
npm install
```

or

```bash
yarn install
```

2) 

```bash
node server/server
```

3)

Use POSTMAN for usage at:

```bash
localhost:1337
```

###### Test Suite

1)

``` bash
npm run test
```

or 

```bash
yarn run test
```
